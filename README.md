# Probability and Statistics for Computer Science - Jupyter Book

An interactive Jupyter Book companion for "Probability and Statistics for Computer Science" by David Forsyth.

## Overview

This repository contains a comprehensive Jupyter Book covering probability and statistics tailored for computer science students. The book emphasizes practical applications, computational approaches, and hands-on examples.

## Building the Book

### Prerequisites

```bash
pip install -r requirements.txt
```

### Build HTML

```bash
jupyter-book build .
```

### Clean Build

```bash
jupyter-book clean .
jupyter-book build .
```
### Live reload using Sphinx
```bash
sphinx-autobuild . _build/html
```
## Book Structure

- **Part I: Describing Datasets** - Visualization and descriptive statistics
- **Part II: Probability** - Probability theory fundamentals
- **Part III: Inference** - Statistical inference and hypothesis testing
- **Part IV: Tools** - Machine learning and data analysis tools

## Original Textbook

Forsyth, D. (2018). *Probability and Statistics for Computer Science*. Springer International Publishing.
